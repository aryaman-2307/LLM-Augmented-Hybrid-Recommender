🏗️ System Architecture
How the hybrid pipeline works
1

MovieLens Data Ingestion

Load the 943×1682 rating matrix from MovieLens 100K, parsing train/test splits (u1–u5). Extract movie metadata including titles and 19-category genre vectors from u.item.

2

SVD Collaborative Filtering

Train a bias-corrected SVD (k=50 latent factors) via scipy.sparse.linalg.svds. Global mean, user bias, and item bias are removed before decomposition. Produces top-50 candidate movies per user.

3

GPT-4o-mini Chain-of-Thought Reasoning

For each candidate, construct a structured prompt with the user's loved/disliked history. GPT-4o-mini reasons step-by-step about thematic alignment, returning a semantic_modifier ∈ [-1, +1] and a natural-language explanation.

4

Hybrid Scoring & Re-ranking

Combine both signals: Final = α × CF_score + β × LLM_modifier. Re-rank candidates by final score. The LLM acts as a semantic refinement layer on top of the statistical baseline.

Final_Score = α × CF_Prediction + β × Semantic_Modifier
CF_Prediction ∈ [1, 5]  |  Semantic_Modifier ∈ [-1, 1]  |  Result clipped to [1, 5]
